Background

What is data race?

  • One of the factors that cause non-deterministic behavior of concurrent programs
    • Unpredictable process scheduling & variations in message latencies
  • 2 types of races that capture different bugs
    • special operations that are provided for safe simultaneous access are called atomic/synchronization operations
    • ordinary operations which are unsafe for simultaneous access are called data operations

Data Race Detection Methods

There are two general approaches to detecting data races: static and dynamic （or on-the-fly）.
In the static approach the source code is being analyzed to detect unserialized accesses to shared data. It may seem like a great idea, but existing techniques are essentially heuristics with very large numbers of false positives due to the fact that static analysis cannot distinguish data dynamically allocated on the heap, etc. Most widely-used data race detection tools use the dynamic approach - at the run time.

Method1: Eraser

The picture below shows the core idea of the “eraser” method.
The core idea of eraser is the “Lockset algorithm”. For each shared variable, eraser maintains a set of candidate locks that protected it for all reads and writes as the program executes. Before any access to shared memory by a thread, eraser checks if it has obtained the required lockset. If the lockset becomes empty, it indicates that there is no lock to protect the variable and a warning will be given. To handle the situation when the above general discipline is violated but the situation is free from data races, such as variable initialization, shared read on data, single-writer, multiple-reader locks, the lockset algorithm is modified and extended.

Method3: Dependence-Aware, Unbounded Sound Predictive Race Detection

  • Incorporates data and control dependence
    • Strong-dependently-precedes
    • Weak-dependently-precedes
  • Handles full program executions
  • Detects more data races than closest related approaches
  • Improves the state-of-the-art performance
