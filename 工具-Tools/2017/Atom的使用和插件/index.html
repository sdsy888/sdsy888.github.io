Atom的使用和插件

Atom的使用

在Atom中使用cmd+,来打开settings

在Atom中使用GitHub

Atom中集成了VSC, git和GitHub. 使用自带的GitHub管理功能可以在Atom中直接修改代码,pull和push代码.

点击Packages-GitHub-Toggle Github会在右侧弹出Github的控制面板, 通过登录GitHub账号获取Token后, Atom的GitHub插件就与你的Gibhub账户建立了联系. 之后再打开的github项目就可以直接再这个面板里pull和push了.

Atom中Git管理下的文件颜色

Atom自带git仓库检测组件，atom会监测当前分支下的文件变动，绿色表示都cmmit了，黄色是有改变没有add. 红色代表新添加没有加入跟踪的文件。

插件

  • markdown-preview

    左右两栏实时预览你写的Markdown

  • linter-markdown

    排查你的Markdown中是否有语法错误

  • markdown-scroll-sync

    可以如丝般顺滑地滚动你的超长Markdown文件

  • markdown-writer

    尤其适合使用Hexo写博客的人,不再命令行和编辑器来回换了!

  • markdown-toc

    专门用于在你的Markdown文件中生成目录TOC

  • markdown-pdf

    不言而喻,只是她还能生成图片

  • tree-view

    • 在左侧显示当前工程的树形结构视图;
    • 在.gitignore文件中设置为忽略的文件夹/文件,会在树形结构中以灰色显示;
    • 如果希望在.gitignore中设置的文件不显示的话,可以在tree-view的设置中勾选Hide VCS Ignored Files

  • atom-beautify

    可以自动美化各种语言, 例如缩进,换行,单行长度等.
    使用前请先pip install autopep8

  • Hydrogen

    可以直接在atom中运行python等程序；

    但是在mac上使用有个小坑需要注意：在python中使用matplotlib画图时候，如果没有手动在程序开头添加%matplotlib inline的话，plt.show画出来的图会以新窗口的形式弹出来，但是却无法手动关闭（现象为手动关闭会卡死）

    解决办法：每次在程序开头手动加入%matplotlib inline或者在hydrogen的设置中的 starup code选项中加入 {"Python 2": "%matplotlib inline"}即可
